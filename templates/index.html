About FooRonto

FooRonto is a web based hybrid restaurant recommendation system. It recommend restaurants to users in Toronto city based on their past visited restaurant history. It uses data from users thar have similar taste in restaurants.

We took the Yelp Dataset. From all the businesses we filtered out all the restaurants that are in Toronto. To give recommendations to users we use Lightfm. LightFM is a Python implementation of a number of popular recommendation algorithms for both implicit and explicit feedback. It is a hybrid recommendation system that combines the features of collaborative filtering and content based filtering.
